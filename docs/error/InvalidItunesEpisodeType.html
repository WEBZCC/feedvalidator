Message

foo is not one of the predefined iTunes episode type

Explanation

itunes:episodeType can only be populated using one of the predefined values "full", "trailer", "bonus", as listed in the specification.

Solution

Change the attribute value to one of the allowed values "full", "trailer", "bonus".
